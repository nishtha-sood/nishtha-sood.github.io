Gender Wage Gap Analysis (20152023)

Closing the Wage Gap: A DEI-Driven Data Dashboard for Employer Brand & Equity Campaigns

Author: Nishtha Sood

Abstract

This project investigates gender-based wage disparities across education levels and occupations using U.S. Census and the Bureau of Labor Statistics.

The dashboard is built to help HR, DEI, and employer branding teams identify pay gaps and prioritize areas for intervention. The goal is to inform equitable hiring, compensation audits, and diversity policies.

Key Business Insights

  • Wage gaps increase with education: Women with graduate degrees earn up to 34% less than men, making this group a key target for upskilling and retention initiatives.
  • High-paying industries have highest gaps: Finance, law, and advisory roles show >35% wage gaps ideal candidates for compensation audits and equity-focused messaging.
  • Post-2020 trends show improvement: The average wage gap has slightly narrowed since 2020, signaling momentum that brands can amplify through storytelling.

Tools & Techniques Used

  • Python: Data cleaning, aggregation, and modeling
  • Pandas & Seaborn: Visualizations (line charts, heatmaps, bar plots)
  • Linear Regression: Modeled wage gap as a function of education and time
  • Data Sources: U.S. Census (ACS), Bureau of Labor Statistics (BLS)

Business Applications

  • HR & Compensation Strategy: Prioritize wage audits by education level or job category
  • DEI Marketing: Use insights to drive data-backed recruitment messaging for women in STEM, finance, or executive leadership
  • Employer Brand Content: Highlight progress in narrowing gaps post-COVID in PR or campaign materials

## Load Education Data

def load_acs_data(years, edu_levels, folder):
    wage_data = []

    for year in years:
        file_path = os.path.join(folder, f"ACSDT1Y{year}.B20004-Data.csv")
        try:
            df = pd.read_csv(file_path, skiprows=1)

            # Try both formats
            for level in edu_levels:
                formats = [
                    f"Estimate!!Total!!Male!!{level}",
                    f"Estimate!!Total:!!Male:!!{level}"
                ]
                male_col = next((col for col in formats if col in df.columns), None)

                formats = [
                    f"Estimate!!Total!!Female!!{level}",
                    f"Estimate!!Total:!!Female:!!{level}"
                ]
                female_col = next((col for col in formats if col in df.columns), None)

                if male_col and female_col:
                    male = df[male_col].iloc[0]
                    female = df[female_col].iloc[0]
                    if male and female and male != 0:
                        wage_data.append({
                            "Year": year,
                            "Education_Level": level,
                            "Male_Earnings": male,
                            "Female_Earnings": female,
                            "Wage_Gap_%": ((male - female) / male) * 100
                        })
        except Exception as e:
            print(f"Failed to load {file_path}: {e}")

    return pd.DataFrame(wage_data)

# Load occupation wage gap table (cleaned version)
occ_df = pd.read_excel(os.path.join(folder, "cpsa2024.xlsx"), sheet_name="cpsaat39", skiprows=4)

# Clean the table
occ_df = occ_df.dropna(subset=["Unnamed: 0", "Unnamed: 4", "Unnamed: 6"])
occ_df = occ_df.rename(columns={
    "Unnamed: 0": "Occupation",
    "Unnamed: 4": "Male_Median_Weekly_Wage",
    "Unnamed: 6": "Female_Median_Weekly_Wage"
})
occ_df["Male_Median_Weekly_Wage"] = pd.to_numeric(occ_df["Male_Median_Weekly_Wage"], errors='coerce')
occ_df["Female_Median_Weekly_Wage"] = pd.to_numeric(occ_df["Female_Median_Weekly_Wage"], errors='coerce')
occ_df = occ_df.dropna(subset=["Male_Median_Weekly_Wage", "Female_Median_Weekly_Wage"])
occ_df["Wage_Gap_%"] = ((occ_df["Male_Median_Weekly_Wage"] - occ_df["Female_Median_Weekly_Wage"]) / occ_df["Male_Median_Weekly_Wage"]) * 100
occ_df = occ_df.sort_values(by="Wage_Gap_%", ascending=False)

from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import LabelEncoder
import numpy as np

# Encode education level to numeric
edu_df_reg = edu_df.copy()
edu_df_reg["Edu_Code"] = LabelEncoder().fit_transform(edu_df_reg["Education_Level"])

# Define X and y
X = edu_df_reg[["Edu_Code", "Year"]]
y = edu_df_reg["Wage_Gap_%"]

# Build model
model = LinearRegression()
model.fit(X, y)

# Output coefficients
intercept = model.intercept_
coeffs = model.coef_

print(f"Intercept: {intercept:.3f}")
print(f"Coef (Education Level): {coeffs[0]:.3f}")
print(f"Coef (Year): {coeffs[1]:.3f}")

Coef (Education Level): 0.318

Average Male vs Female Earnings Over Time

While the gender wage gap % tells us the proportional disparity, this chart shows actual earnings. Male earnings consistently outpace female earnings, though both have increased modestly over time. Notably, the absolute gap has not narrowed significantly, reinforcing the persistence of pay inequity despite educational gains and social progress.

Insight from Regression

    Year Coefficient = -0.358: Wage gaps decrease slowly over time (~0.36% per year).

The regression supports using education level as a targeting variable in campaigns that promote fairness, mentorship, or equal advancement opportunities.

Key Takeaways & Recommendations

  • Higher Education = Higher Gap: Women with bachelor's or graduate degrees still face >30% wage disparities.
  • Worst Gaps in Finance & Law: Occupations with discretionary pay show highest gaps.
  • Standardized Roles Show More Equity: Government, unionized, and service-based jobs show smaller wage gaps.

Recommendations:

  • Enforce wage transparency laws in private sectors.
  • Encourage salary audits and equitable promotion structures.
  • Normalize flexible work policies tied to pay equity metrics.

Addressing the gender wage gap requires systemic change not just individual education or career choices.
